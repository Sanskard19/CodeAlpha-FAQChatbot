# NexusAI FAQ Chatbot

A smart FAQ chatbot that uses **TF-IDF vectorization** and **cosine similarity** to match user questions with the most relevant FAQ and return the best answer.

## Features

- **40 high-quality FAQs** about a fictional SaaS product (NexusAI)
- **Pure JavaScript NLP pipeline** — no backend required (TF-IDF + cosine similarity implemented from first principles)
- **Live NLP visualization** — shows tokenization, stopword removal, TF-IDF, similarity ranking
- **Chat UI** with confidence scores and related questions
- **Light/Dark mode** + mobile responsive

## How to Run

1. Clone this repo
2. Open `faq-chatbot.html` in any browser
3. Ask anything about NexusAI

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **NLP:** TF-IDF vectorization, cosine similarity (custom JS implementation)
- **No frameworks:** Zero dependencies
